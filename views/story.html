What We Offer

Scratch

With Scratch (a project of MIT Media Lab) our campers learn the fundamentals of programming and problem solving by snapping together colorful blocks of code to create interactive stories and games.


Java

With Java our students learn the essentials of object oriented programming and and get a taste for the limitless possibilities of a full professional programming language.


Tiny Groups

We’ve always loved taking small classes and know that they’re the best way to learn, so we guarantee that our camp will have a 10-1 student-teacher ratio or better. Students will get the one-on-one help they need and progress at the best pace for them.


Project Centered

The core of our curriculum is a sequence of 15 Scratch projects. For each project students have creative freedom to build whatever they think is fun while integrating new skills and concepts. Towards the end of the camp students build a capstone project like Pong or Mario and then follow a second series of Java projects.
